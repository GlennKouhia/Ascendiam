All Rise with Better Software.

Welcome to Ascendiam (TM), a hobby site that's used to explore responsive web development, fast full screen video delivered for free via a content delivery network, and software as a service applications.

Plan of Record, V0.1

Within the Web Apps navigation tab you'll find links to responsive pages that are web applications. After some research and coding I intend for the apps to do several things, like

    Define and build a custom database, then populate it with data called from cloud service APIs;
    Mock up an ecommerce product selection shopping page;
    Present survey questions, capture user answers, present survey insights.
    Add banner advertising to a live page.
    Summarize this site's traffic data (GA tags) and AdWord performance (advertising insights).
    Link to a short live video shot at the 2019 MLS final from an iPhone 6S.


More to come

This site will evolve. So, check back soon. I hope to have more to show than simple code clean ups (e.g. html validation and CSS linter validation).
